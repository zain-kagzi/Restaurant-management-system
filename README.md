🍽️ Restaurant Management System

A full-stack web application to manage restaurant operations like orders, menu, and billing.

🚀 Features
- Admin dashboard to manage menu items
- Order management system
- Customer billing system
- Responsive UI

🛠 Tech Stack
- Frontend: HTML, Tailwind CSS,JS
- Backend: PHP , AJAX
- Database: MySQL
